## 电子相册系统开发文档

### 1 技术选型

前端：Vue3.0 + Element-Plus

后端：django框架

数据库：mysql

环境配置

前端：node.js配置、npm配置、vue环境配置

前端依赖包，通过npm包管理器安装（不配置代理的话可能会慢，可以用cnpm）

```
npm install element-plus --save
npm install sass -D
npm install vue-router@4
```
